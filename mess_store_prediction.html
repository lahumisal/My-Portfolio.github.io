Mess Store Prediction System And Message Accommodation System

A system that utilizes linear regression algorithm with Weka library to predict and maintain store inventory records, and sends automated emails to suppliers for purchasing required products.

About the Project

we maintain record of store inventory and utilize linear regression algorithm with the help of Weka library because it is popular open source machine learning library in java. And we send mail to the supplier for purchase the product which is required in the store in the future.

Features

  • User Authentication: Designed and implemented secure user registration, login, and logout functionalities using JWT. Ensured robust error handling and token management with a 99.9% uptime.
  • Reduce Paperwork: Transition from manual record-keeping to a digital system.
  • Predictive Analysis: Store historical price data and analysis it.
  • Inventory Management: Track inventory levels, purchases, and usage to maintain optimal stock levels.
